Sistem minimal
------------------
- Microsoft Windows NT keatas
- RAM 128 keatas
- Pentium III atau setara

Introduction
--------------
SigmaAV adalah Antivirus Freeware dan OpenSource yang bebas untuk dikopi dan disebarluaskan kesiapa saja dan dimana saja. Untuk itu, saya mohon untuk tidak memberikan harga pada software ini. Karena OpenSource, saya akan membagikan source code untuk program ini gratis, jika ingin meminta, silahkan kirim e-mail ke alamat xtfcomp@gmail.com. Dibuat menggunakan Delphi 7.0 dengan beberapa komponen tambahan seperti tema tampilan SUI Theme Pack yang bisa anda cari di Internet.

Fitur - fitur 
-------------
a. Registry editor
  - Membantu anda dalam mengembalikan fungsi - fungsi registry yang diblok oleh virus.
b. Process Killer
  - Seperti Task Manager yang membantu anda jika Task Manager Windows diblokir
c. Port Information
  - Memberitahu anda koneksi aktif yang digunakan
d. Menggunakan 3 type pencarian
    1. CRC32
	Pencarian menggunakan hasil checksum yang telah didefinisikan sebelumnya di database
    2. HEX
	Pencarian menggunakan karakter tertentu dari sebuah file virus
    3. MD5
	Pencarian menggunakan HASH dari sebuah file
e. Scan Options
    1. Max file Scan : Batas maksimal file yang akan discan
    2. Min file Scan : Batas minimal file yang akan discan
    3. Heuristic : Penggunaan metode heuristic dalam pencarian, membantu mengenali virus - virus baru.
    4. Always start memory scanning : Selalu menscan memory terlebih dahulu
    5. Place shortcut scanning : meletakan shortcut di explorer, pada menu klik kanan
f. Skip the file...
   - Membantu untuk melompati file yang dirasa discan terlalu lama.
g. Shutdown when complete
   - Jika penscanan sudah selesai, otomatis, file yang terinfeksi akan di repair, jika gagal, akan dimasukan ke dalam Quarantine Room, kemudian komputer akan shutdown.
